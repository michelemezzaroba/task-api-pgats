# Task API

API desenvolvida como trabalho final da disciplina PGATS. com foco em autenticação, gerenciamento de tarefas e testes automatizados integrados em pipeline CI.


## Tecnologias
- Node.js
- Express
- MongoDB
- GraphQL
- Mocha
- Chai
- Supertest
- Sinon

## Rodar projeto

npm install  
npm start  

## Rodar testes

npm test
